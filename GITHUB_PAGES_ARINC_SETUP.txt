重要：GitHub Pages 是純靜態網站，_worker.js 不會執行。
ARINC 自動更新必須依靠 GitHub Actions。

部署後請在 GitHub 儲存庫確認存在：
.github/workflows/update-arinc.yml

若用 GitHub 網頁手動上傳檔案，macOS Finder 可能隱藏 .github 資料夾，造成工作流程漏傳。
請在 GitHub 儲存庫內按 Add file > Create new file，檔名輸入：
.github/workflows/update-arinc.yml

再把根目錄 INSTALL_update-arinc-workflow.yml 的全部內容貼入並 Commit。
之後到 Actions > Update ARINC Pacific HF > Run workflow 手動執行一次。
成功後 data/arinc.json 會產生新提交，GitHub Pages 隨後更新。
